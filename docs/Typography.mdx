# Typography

Reference: https://patternlab.muenchen.space/?p=viewall-guidelines-typography

## Headings

For all Headings just use the semantic h1 to h5 HTML Elements - they will automatically be styled according to the MDE Design System.

## Body

For the body there are three different sizes. Each with it's own class to be used.

For all sizes the additional class <pre>mde-bold</pre> can be used to make the text bold.
